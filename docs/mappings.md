# UI
Control Menu:
- Home
- Reset Scene
- Simulate
- Open Content Drawer

Content Drawer:
- All Assets

Object Editor Menu:
- Edit Object Properties
    - Scale
    - Rotation
    - Location
- Delete Object
- Mass
- Color
- Gravity (on/off)
- duplicate object



## Draw Mode
Middle Button: Hold to Draw
Front: Grab and Move drawings
Rear: undo drawing, hold to clear all drawing

## Edit Mode (Default)
Front: Select, Hold to Grab an Object


X: Open/Close Control Menu
